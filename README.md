# Identificador de música

Este projeto é um **identificador de música**, desenvolvida com **HTML, CSS e JavaScript**, com o objetivo de treinar **lógica de programação**, **funções**, **condições** e **consumo de API**.
---
##  Lógica do projeto

- Usuário clica → iniciarGravacao()
- Pede microfone
- Grava por 8 segundos
- Junta os pedaços do áudio
- Envia para API
- API responde com nome da música
- Mostra na tela

## Conceitos importantes que foram usados 
- Async / Await: Resolve o problema pois algumas coisas demoram a acontecer como: Pedir o microfone, Chamar a API e buscar os dados da internet
  (Promises é uma coisa que irá acontecer)
- MediaRecorder: É uma ferramenta do navegador que grava áudio ou vídeo.
- Blob(Binary Large Object): Arquivo temporário;
- Array: É uma lista. (Foi definida uma Array inicialmente vazia para guardar os trechos dos áudios)
- Fetch API: É a forma moderna de pedir dados da internet.
- Promise (.then): Promessa de que algo vai acontecer.
- DOM Manipulation: DOM é o HTML transformado em objeto JavaScript.
